PTP FO Deployment at Balai Pelatihan Kesehatan Yogyakarta

09 Jan 2026
By Rosyid Majid

Project Overview

Balai Pelatihan Kesehatan Yogyakarta required a reliable point-to-point fiber optic connection between the Server in the East Building and the Staff Room in the West Building to enable smooth data communication between the two buildings.

The project included HTB installation, fiber optic cabling, Box Panel setup, and Switch deployment, providing connectivity for PCs, laptops, and a Unifi Access Point.

Deployment Scope

  • PTP FO Link: Server East Building <=> Staff Room West Building
  • Installed Equipment:
    • HTB at both Server and Staff locations
    • Box Panel at Staff Room West Building
    • Switch for PCs, laptops, and 1 Unifi AP
  • Cabling: Single-mode fiber optic for the PTP link
Location Documentation
Server East Building
Staff Room West Building
Fiber Attenuation Test

Responsibilities

  • HTB Installation: Mounted HTB units at both Server and Staff ends to secure fiber termination.
  • PTP FO Cabling: Pulled and terminated fiber optic cable between buildings.
  • Box Panel Setup: Installed and organized fiber connections at the Staff Room.
  • Network Validation: Tested connectivity and verified stable data transmission across the link.
  • Stakeholder Coordination: Coordinated with the assigned Person in Charge (PIC) regarding device placement, fiber optic cable routing, and on-site technical requirements to ensure proper installation and compliance with building infrastructure.

Implementation Details

Physical Installation

  • Installed HTB securely at both the Server and Staff locations.
  • Mounted Box Panel for organized fiber termination in the Staff Room.

Fiber Optic Deployment

  • Pulled single-mode fiber optic between the two buildings.
  • Terminated and tested the fiber link for low-loss connectivity.

Network Equipment Configuration

  • Installed switch for wired clients and Unifi AP.
  • Verified full network connectivity and stable communication between devices.

Tools & Technologies

  • Single-Mode Fiber Optic (SM FO)
  • Home Terminal Box (HTB)
  • Box Panel
  • Network Switch
  • Ethernet cabling for client devices

Network Analysis

  • Reliability: PTP FO provides a stable high-speed connection between buildings.
  • Scalability: The design allows future expansion for additional devices.
  • Operational Efficiency: Structured cabling and organized termination simplify troubleshooting.

Outcome

  • Successfully established a PTP FO link between the Server East Building and Staff Room West Building.
  • Achieved stable and reliable network connectivity for PCs, laptops, and the Access Point.
  • Demonstrated hands-on experience in fiber optic deployment and inter-building networking.
