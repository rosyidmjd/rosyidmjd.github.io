Instalasi PTP FO di Balai Pelatihan Kesehatan Yogyakarta

09 Jan 2026
By Rosyid Majid

Gambaran Proyek

Balai Pelatihan Kesehatan Yogyakarta membutuhkan koneksi fiber optik point-to-point (PTP FO) yang andal antara Server di Gedung Timur dan Ruang Staff di Gedung Barat untuk mendukung komunikasi data yang lancar antar gedung.

Proyek ini mencakup instalasi HTB, penarikan dan terminasi kabel fiber optik, pemasangan Box Panel, serta deployment Switch, guna menyediakan konektivitas untuk PC, laptop, dan satu unit UniFi Access Point.

Ruang Lingkup Pekerjaan

  • Link PTP FO: Server Gedung Timur <=> Ruang Staff Gedung Barat
  • Perangkat Terpasang:
    • HTB di sisi Server dan sisi Staff
    • Box Panel di Ruang Staff Gedung Barat
    • Switch untuk PC, laptop, dan 1 UniFi AP
  • Media Kabel: Fiber optik single-mode untuk link PTP
Lokasi Dokumentasi
Server Gedung Timur
Ruang Staff Gedung Barat
Pengujian Redaman Fiber

Tanggung Jawab

  • Instalasi HTB: Pemasangan HTB di sisi Server dan Staff untuk terminasi fiber optik.
  • Penarikan Kabel PTP FO: Penarikan dan terminasi kabel fiber optik antar gedung.
  • Pemasangan Box Panel: Instalasi dan penataan koneksi fiber di Ruang Staff.
  • Validasi Jaringan: Pengujian konektivitas dan kestabilan transmisi data.
  • Koordinasi Stakeholder: Berkoordinasi dengan Person in Charge (PIC) terkait penempatan perangkat, jalur kabel fiber optik, serta kebutuhan teknis di lokasi agar instalasi sesuai dengan infrastruktur gedung.

Detail Implementasi

Instalasi Fisik

  • HTB dipasang dengan aman di lokasi Server dan Ruang Staff.
  • Box Panel dipasang untuk memastikan terminasi fiber tertata dan rapi.

Deployment Fiber Optik

  • Penarikan kabel fiber optik single-mode antar dua gedung.
  • Terminasi dan pengujian link untuk memastikan koneksi low-loss dan stabil.

Konfigurasi Perangkat Jaringan

  • Pemasangan switch untuk klien kabel dan UniFi Access Point.
  • Verifikasi konektivitas jaringan dan komunikasi antar perangkat.

Tools & Teknologi

  • Fiber Optik Single-Mode (SM FO)
  • Home Terminal Box (HTB)
  • Box Panel
  • Switch Jaringan
  • Kabel Ethernet untuk perangkat klien

Analisis Jaringan

  • Keandalan: PTP FO memberikan koneksi berkecepatan tinggi dan stabil antar gedung.
  • Skalabilitas: Desain jaringan mendukung penambahan perangkat di masa depan.
  • Efisiensi Operasional: Penataan kabel dan terminasi yang rapi mempermudah troubleshooting.

Hasil Akhir

  • Berhasil membangun jaringan PTP FO antara Server Gedung Timur dan Ruang Staff Gedung Barat.
  • Menghasilkan koneksi jaringan yang stabil dan andal untuk PC, laptop, dan Access Point.
  • Menunjukkan pengalaman langsung dalam deployment fiber optik dan jaringan antar gedung.
